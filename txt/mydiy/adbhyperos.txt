ADB卸载小米预装系统应用（Hyper OS 2）

MIUI应该也部分适用

PS：为了省电和流畅而卸载多余APP的前提是：这些APP在后台运行，不然卸了也没什么效果，反而会引发Bug。所以应该参考后台运行有选择地卸载多余APP。

查看后台运行最简单的办法就是使用ADB命令打开安卓系统自带的服务（adb shell am start -n com.android.settings/com.android.settings.RunningServices），其中正在运行中的服务日常耗电，已缓存的进程基本不耗电。

特别说明

1.如需保留应用数据及缓存卸载，adb命令替换为

   adb shell cmd package uninstall -k <包名>

2.恢复出厂预设应用的adb命令，不过我还是建议用scene3.7.3免费版恢复，简单嘛

    adb shell cmd package install-existing <包名>

3.启用已冻结应用的adb命令(上条适用于无root权限情况，但无法解冻系统应用。下条为使用root权限进行应用解冻)

    adb shell pm enable <包名>

    adb shell su -c "pm enable <包名>"

ps：冻结应用的adb命令(上条适用于无root权限情况，但无法冻结系统应用。下条为使用root权限进行应用冻结)

   adb shell pm disable-user <包名>

   adb shell su -c "pm disable <包名>"

4.除非刷了官改系统或修改过system/framework中的services.jar文件，否则卸载会卡米的应用有：应用商店(com.xiaomi.market，MIUI必卡米，hyperos似乎不会卡米)、小米账户(com.xiaomi.account)、小米云服务(com.miui.cloudservice)、小米云备份(com.miui.cloudbackup) 、系统更新（com.android.updater，概率卡米）、查找设备(com.xiaomi.finddevice，概率卡米)、手机管家、系统界面、Android系统、html查看器、网络管理。

如果非要卸载卡米应用，解锁BL是大前提，修改services.jar文件具体做法参考 https://www.zhihu.com/question/265117194

PS：所谓卡米就是开机之后屏幕只显示小米的logo进不了系统。以上卡米应用未实测，仅收集各路消息作汇总防止踩雷，尝试前记得备份好数据。因为如果不小心卡米了，还没解BL，那只有进recovery重置系统一条路了，所有的数据都会消失。。。去售后也是帮你重置系统，救不了数据

5.打开屏幕旋转建议的adb命令，偶尔需要搭配HyperCeiler中强制开启旋转建议使用

adb shell settings put secure show_rotation_suggestions 1
