#1.卸载多余应用

使用Windows+i快捷键进入“设置”页面，选择左侧的“应用”栏目，点击“安装的应用”选项，界面所示的就是电脑上能检测到的已经安装的应用（部分应用虽然安装了但是未必能在这里显示）

接下来介绍软件卸载方法（以HP Smart为例）：

以下应用推荐大家直接卸载：

Clipchamp-视频编辑器、Cortana、Microsoft OneDrive、Microsoft ToDo、Microsoft 咨询、Solitaire Collection、Xbox、Xbox Live、地图、人脉、天气、HP Smart

注意：如果你不知道上面说的应用是什么，大概率是根本用不到的，可以直接卸载。如果你能用到以上应用，可以保留。

#2.开启高性能模式

选择左侧的“系统”选项，点击“电源和电池”选项

电源模式建议选择“最佳性能”

#3.更改视觉显示效果

选择左侧的“系统”选项，点击“系统信息”选项

按照以下步骤一次操作即可，铜鼓去掉一些没必要的效果，可以提高系统响应速度

#4.更改资源管理器显示

打开资源管理器，删除左侧基本不用的快速访问，添加自己常用的文件夹到快速访问栏目

按照如下步骤操作，“单击项目”的方式根据自己的习惯选择双击还是单击，其他的建议按照我的设置来

#5.优化系统搜索文件速度

打开资源管理器，按“F11”按钮进入全屏，再点击“F11”按钮退出全屏，在右上角搜索框中搜索本地文件的效率极大提高。

而且无论文件夹中有多少文件，每次点击都是秒开

#6.内存优化大法

由于内存速度远远大于固态硬盘，更大于机器硬盘，但是windows默认的内存缓存只有8MB，接下来就解除这个8MB的限制，增大内存缓存，提高软件运行速度。

使用“windows+R”快捷键，输入“regedit”，点击“确定”按钮进入注册表

按照以下目录，进入注册表中进行设置

计算机\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Memory Management

双击“DisablePagingExecutive”选项，按照以下设置

双击“LargeSystemCache”选项，同上设置（轻薄本可能没有这个选项）

操作完成后重启电脑，在搜索框中输入“windows powershell”，选择“以管理员身份运行”，

进入命令行窗口后输入“mmagent”命令，查看“MaxOperationAPIFiles”的数值

然后输入“set-mmagent”命令，输入8192（前后不要输入任何空格），按回车键。修改完成

再次输入“mmagent”命令，查看“MaxOperationAPIFiles”的数值，会显示修改后的数值

#8.设置定时清理垃圾文件

使用Windows+i快捷键进入“设置”页面，选择左侧的“系统”选项，点击“存储”选项，

打开“存储感知”

并点击由此的小箭头进入设置界面，接下来的三项设置根据自己的需求设置即可，点击“立即运行存储感知”按钮

回到刚刚的“存储”界面，点击“清理建议”

这里会罗列一些可以清理的文件，感觉需求清理即可

#9.关闭系统广告

回到设置页面，选择左侧的“隐私和安全性”选项，点击“常规”选项，

建议全部关掉

回到设置页面，选择左侧的“隐私和安全性”选项，点击“诊断和反馈”选项，

这里还涉及一点自己电脑隐私数据的问题，这里一并说了，按照我的设置即可

#10.优化系统任务栏

回到设置页面，选择左侧的“个性化”选项，点击“任务栏”选项

建议关闭小组件，只显示“搜索”图标，其他的根据自己的需求设置即可

#11.优化搜索页面

点击任务栏的“搜索”按钮，点击右上角的三个点，点击“搜索设置”选项，

建议直接关闭“显示搜索要点”选项，可以去除搜索栏目的无用信息个广告


“云搜索内容”根据自己的需求选择，我基本只用google搜索，这里就全部关掉了

#12.关闭系统不必要通知

回到设置页面，选择左侧的“系统”选项，点击“通知”选项

接下来根据自己的需要可以直接关闭所有应用的通知按钮，也可以选择性关闭个别应用的通知功能

#13.管理系统右键功能

因为电脑上安装的软件不同，个别软件回个右键中添加太多无用的功能，这些功能太多的情况下会极大影响使用体验，如下图所示，这里建议使用“ContextMenuManager”这款开源软件来对系统右键功能进行管理。软件下载网址如下：

https://github.com/BluePointLilac/ContextMenuManager/releases/tag/3.3.3.1


Windows 7系统下载第一个，Windows 8及以上系统下载第二个。这款应用是不用安装的，直接双击就可以运行，根据自己的需要设置系统右键功能即可。

#15.系统常用快捷键

15.1 Ctrl家族 (基础编辑与导航)

Ctrl + C: 复制选中的内容（文本、文件、文件夹等）。

Ctrl + X: 剪切选中的内容（文本、文件、文件夹等）。

Ctrl + V: 粘贴剪贴板中的内容。

Ctrl + Z: 撤销上一步操作。

Ctrl + Y: 恢复被撤销的操作（重做）。

Ctrl + A: 全选当前窗口或文档中的所有内容。

Ctrl + S: 保存当前文档或文件。

Ctrl + F: 打开“查找”对话框（在文档、网页或文件资源管理器中搜索）。

Ctrl + N: 新建（通常是新建窗口、文档或文件，具体取决于当前应用）。

Ctrl + P: 打印当前文档或页面。

Ctrl + W: 关闭当前窗口或标签页（在浏览器或文件资源管理器中）。

Ctrl + Tab: 在应用程序内的多个标签页之间切换（如浏览器、文件资源管理器）。

Ctrl + Shift + Tab: 在应用程序内的多个标签页之间反向切换。

Ctrl + Esc: 打开或关闭“开始”菜单（功能上等同于单独按 `Win` 键）。

Ctrl + Shift + N: 在文件资源管理器中新建文件夹。

Ctrl + 鼠标滚轮: 在文件资源管理器或桌面上调整图标大小；在大多数应用程序中缩放视图（放大/缩小）。


15.2 Alt家族 (窗口控制与菜单导航)

Alt + Tab: 在打开的应用程序窗口之间切换（按住 Alt 不放，重复按 Tab 键选择）。

Alt + Shift + Tab: 在打开的应用程序窗口之间反向切换。

Alt + F4: 关闭当前活动窗口或应用程序。如果当前是桌面，则打开关机对话框。

Alt + Enter (在文件资源管理器中选择文件/文件夹时): 显示所选项目的属性对话框。

Alt + Space: 打开当前活动窗口的控制菜单（最小化、最大化、关闭等）。

Alt + 字母: 激活当前应用程序菜单栏中带下划线的对应字母命令（需要菜单栏可见）。例如在文件资源管理器中按 `Alt + F` 打开“文件”菜单。

Alt + ←` (左箭头): 在文件资源管理器或浏览器中后退。

Alt + →` (右箭头): 在文件资源管理器或浏览器中前进。

Alt + ↑` (上箭头): 在文件资源管理器中打开当前文件夹的上一级文件夹。

Alt + D: 将焦点定位到文件资源管理器或浏览器的地址栏（方便输入或复制路径/网址）。


15.3 Windows键家族 (系统级操作)

Win` (单独按下)**: 打开或关闭“开始”菜单。

Win + D: 显示桌面（最小化所有窗口）。再按一次恢复窗口。

Win + E: 打开文件资源管理器。

Win + L: 锁定计算机（快速切换用户或进入登录界面）。

Win + R: 打开“运行”对话框（用于快速运行命令，如 `cmd`, `calc`, `notepad`）。

Win + I: 打开Windows设置。

Win + A: 打开操作中心（通知中心和快速设置）。

Win + S` 或 `Win + Q: 打开Windows搜索（搜索文件、应用、设置、网页等）。

Win + V: 打开剪贴板历史记录（需要Windows 10 1809或更高版本并启用此功能）。

Win + .` (句点) 或 `Win + ;` (分号)**: 打开表情符号和符号面板。

Win + Tab: 打开任务视图（查看所有虚拟桌面和当前打开的窗口）。

Win + Ctrl + D: 创建新的虚拟桌面。

Win + Ctrl + →/←: 在虚拟桌面之间切换。

Win + Ctrl + F4: 关闭当前虚拟桌面。

Win + 数字键 (1-9): 打开或切换到任务栏上对应位置（从左数第1-9个）固定的应用程序。如果程序已打开则切换到它。

Win + Shift + 数字键 (1-9): 以管理员身份打开任务栏上对应位置（从左数第1-9个）固定的应用程序的新实例。

Win + +: 打开放大镜并放大。

Win + -: 使用放大镜时缩小。

Win + Esc: 关闭放大镜。

Win + P: 更改投影模式（仅电脑屏幕、复制、扩展、仅第二屏幕）。

Win + X: 打开“快速链接”菜单（包含设备管理器、磁盘管理、命令提示符/Windows终端、电源选项等常用管理工具入口）。

#16. 卸载系统广告推送器

「Windows Web 体验包」全称Windows Web Experience Pack，是Windows 11 中的一个系统组件。 它主要负责为「小组件」这类功能，用于展示天气、新闻、股票等动态信息，其实就是个广告推送器。根据电脑 Windows 系统的隐私和安全设置，这垃圾东西最爱要我的位置信息。小组件这玩意我自己是不用的，直接卸载掉！

卸载方法如下：

① 在应用菜单开始栏搜索 cmd，以管理员身份运行

② 输入以下代码：引号必须要是英文的

winget uninstall "Windows Web Experience Pack"

③ 等待卸载完成
